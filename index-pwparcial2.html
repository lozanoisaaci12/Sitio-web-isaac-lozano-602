Tarea 2: Maquetación de una página web

Copia y pega los códigos de tus archivos HTML Y CSS en un documento en blanco, también toma captura de pantalla a la visualización de tu maquetación en un navegador. Tu maquetación debe contener los segmentos de Encabezado (dividido en tres segmentos), Menú de navegación, Main (dividido en tres partes), y Pie de página.

Tarea 3: Propiedades de CSS para el Menú de navegación.

Lee el archivo publicado en Classroom sobre las propiedades del lenguaje CSS e identifica las propiedades que te permiten darle un buen formato a los elementos que contiene el Menú de Navegación. Copia y pega tus códigos HTML y CSS en un documento de texto, así también toma captura de tu página en la que se visualice el formato aplicado al Menú de navegación.
